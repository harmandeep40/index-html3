HTML Tags

<p>
The <p> tag defines a paragraph. It is used to group blocks of text into distinct paragraphs.
<a>
The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
<img>
The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed.
<ul>
The <ul> tag defines an unordered list. Items in the list are typically displayed with bullet points.
<table>
The <table> tag defines a table in an HTML document. It is used in conjunction with the <tr> (table row), <th> (table header), and <td> (table data) tags to structure the table.
